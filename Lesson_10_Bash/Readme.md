# Занятие 10. Bash

## Домашнее задание
Пишем скрипт

**Цель:**
Написать bash-скрипт, который ежечасно формирует и отправляет на email отчёт о работе веб-сервера;

Что нужно сделать?

Написать скрипт для CRON, который раз в час формирует отчёт и отправляет его на заданную почту.

Отчёт должен содержать:
- IP-адреса с наибольшим числом запросов (с момента последнего запуска);
- Запрашиваемые URL с наибольшим числом запросов (с момента последнего запуска);
- HTTP-коды ответов с указанием их количества (с момента последнего запуска);
- Ошибки веб-сервера/приложения (с момента последнего запуска).

Скрипт должен предотвращать одновременный запуск нескольких копий, до его завершения.

В письме должен быть прописан обрабатываемый временной диапазон.

Отобрать IP-адреса, посчитать сколько раз они повторяются, отсортировать по убыванию, оставить для отчета топ 10 адресов.
Вычленить URL, посчитать сколько раз они повторяются, отсортировать по убыванию, оставить для отчета топ 10 URL.
Вычленить HTTP-коды ответов, посчитать сколько раз они повторяются, отсортировать по убыванию, оставить для отчета топ 10 HTTP-кодов ответов.
Вычленить ошибки - про какие ошибки идет речь? Целиком строки с ошибками в отчет пихнуть?.

Отчет - один файл или на каждый пункт отдельный? Каждый час новый за прошедший час.

Сформировать задание для cron для запуска скрипта каждый час.
